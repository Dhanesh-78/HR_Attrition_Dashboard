# HR Employee Attrition Dashboard

## Project Overview
This project analyzes employee attrition trends using SQL, Python, and Tableau.  
The goal of the project is to identify the major factors affecting employee turnover and build an interactive dashboard for HR insights.

The dashboard helps analyze:
- Overall employee attrition
- Department-wise attrition
- Job role attrition
- Overtime impact on attrition
- Age group analysis

---

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- SQL (MySQL)
- Tableau

---

## Dataset Information
The dataset contains HR employee records including:
- Department
- Job Role
- Age
- Monthly Income
- Overtime
- Attrition status
- Job Satisfaction
- Employee demographics

---

## Key Insights
- Overall attrition rate is **16.1%**
- A total of **237 employees** left out of **1470 employees**
- The **Sales department** has the highest attrition rate (**21%**)
- **Sales Representatives** show the highest attrition (**40%**)
- Employees working overtime have much higher attrition (**30.5%**) compared to employees without overtime (**10.4%**)
